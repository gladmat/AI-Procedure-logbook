# Replit Agent Prompt: Add Soft Tissue Trauma Procedures & Diagnoses

## Summary

Add soft tissue trauma coverage to the procedure and diagnosis picklists. Currently the app handles facial fractures, hand trauma, and complex orthoplastic wounds well, but has no home for simple-to-moderate laceration repairs — the bread-and-butter of plastics on-call work. This adds:

- **8 new procedures** in Head & Neck → new subcategory "Facial Soft Tissue Trauma"
- **6 new procedures** in General → new subcategory "Soft Tissue Trauma"
- **6 new diagnoses** in Head & Neck → new subcategory "Facial Soft Tissue Trauma"
- **4 new diagnoses** in General → new subcategory "Soft Tissue Trauma"
- **1 update** to existing orthoplastic pretibial laceration diagnosis (add repair procedure suggestion)

Total new procedures: 14 (some cross-tagged, so will appear in multiple specialties)
Total new diagnoses: 10

## SNOMED CT Notes

**Procedure codes used:**
| Code | Display | Used For |
|---|---|---|
| 238182008 | Suture of skin (procedure) | Simple laceration repairs — all sites. Body site post-coordinated via case metadata. |
| 20720004 | Wound repair (procedure) | Complex/layered repairs, degloving management, traumatic skin loss closure. |
| 360160003 | Exploration of wound (procedure) | Wound exploration + debridement entries. **VERIFY** on SNOMED browser. |

**Diagnosis codes used:**
| Code | Display | VERIFY? |
|---|---|---|
| 284009009 | Laceration of face (disorder) | No — well-established concept |
| 283682007 | Laceration of nose (disorder) | VERIFY |
| 283679001 | Laceration of lip (disorder) | VERIFY |
| 284003008 | Laceration of eyelid (disorder) | VERIFY |
| 283545005 | Laceration of ear (disorder) | VERIFY |
| 262560009 | Laceration of scalp (disorder) | VERIFY |
| 283681000 | Laceration of skin (disorder) | No — generic laceration concept |
| 283680004 | Laceration of pretibial skin (finding) | No — already in use in orthoplastic |
| 262561008 | Avulsion injury of skin (disorder) | VERIFY |
| 282026001 | Traumatic loss of skin (finding) | VERIFY |

**Design rationale for using 238182008 (Suture of skin) across multiple entries:**
All simple laceration repairs share the same core procedure — suturing skin. Body site differentiation comes from case-level metadata (diagnosis + body region), not from the procedure code. This follows the project's established pattern where skin excision procedures (e.g., "BCC excision — face" vs. "BCC excision — trunk") share the same SNOMED procedure concept with site post-coordinated. The simple/complex split uses a different concept (20720004 Wound repair) because layered closure is a fundamentally different procedure type, matching OPCS-4's S41/S42 distinction.
